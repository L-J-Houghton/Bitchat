                      README:
--------------------------------------------------------------

Here you will find the files related to the 
web app for the Bitchat project appropriately
named. The 'OTHER' folder contains other files
that may be of relevance to the poroject. These
include the font license which is an open font
license (OFL) and the SQL file used to construct 
the database for both the web and android platforms
of this project. The web app can be visited using the
link below and the android app is available via
the Google playstore and can be found in the link 
below also:
---------------------------------------------------------------
                 Bitchat (Android):
---------------------------------------------------------------
https://play.google.com/store/apps/details?id=com.bitchat&hl=en
---------------------------------------------------------------
                  Bitchat (Web app):
---------------------------------------------------------------
http://doc.gold.ac.uk/~lhoug001/bitchat/home.php
